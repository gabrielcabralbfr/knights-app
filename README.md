# Knights application

## Dependencies

1. Install [Docker](https://docs.docker.com/)
1. Install [Node and NPM](https://nodejs.org/en/)


## Running the application

1. With docker
Run the following command to startup the application

```bash
docker-compose up -d
docker-compose start
```

2. With NPM
On root folter
```bash
npm start
```
Open another terminal
```bash
cd views/knights-ui
npm start
```
